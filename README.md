This is a minimal scheduler written with sched-ext.

```bash
# build
sh ./build.sh

# run
sh ./run.sh

# do something ...

# stop
sh ./stop.sh
```

You can read more at [my blog](https://mostlynerdless.de/blog/2024/10/22/writing-a-minimal-linux-scheduler-in-c-with-sched-ext/).
